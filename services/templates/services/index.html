Automated Phishing Tool

The CSIR Automated Phishing Tool online web services that allow a user (usually a manager) to add a list of employee addresses on to the site. Once addresses have been added, the system will send phishing email to the list of recipients. The email will include a URL that will have an encrypted (md5) key embedded in it. This key is what will be used to identify the different users (phishing victims).
